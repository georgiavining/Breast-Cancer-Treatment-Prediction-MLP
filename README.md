# Breast Cancer Treatment Response Prediction — MLP

A deep learning project that predicts **relapse-free survival time** for breast cancer patients prior to chemotherapy, using clinical data and MRI-derived imaging features. Built as part of MSc Machine Learning in Science coursework.

---

## Overview

This project implements a **Multilayer Perceptron (MLP)** regression model trained on a combination of structured clinical data and quantitative MRI features. The target variable is **relapse-free survival time** — a clinically meaningful outcome that reflects how long a patient remains disease-free following treatment.

Four experimental runs were conducted, each using a different data strategy, to systematically evaluate the impact of augmentation and oversampling on model performance.

---

## Motivation

Breast cancer treatment response varies significantly between patients, and standard chemotherapy regimens do not work equally well for everyone. Predicting relapse-free survival time *before* treatment begins could allow clinicians to:

- Tailor chemotherapy protocols to individual patients
- Identify high-risk patients who may benefit from more aggressive intervention
- Reduce unnecessary treatment burden for patients likely to respond well

This project explores whether machine learning — trained on pre-treatment clinical and imaging data — can meaningfully support these decisions.

---

## Dataset

The model is trained on a dataset combining two sources of patient information:

| Feature Type | Description |
|---|---|
| **Clinical data** | Patient demographics, receptor status (ER, HER2), gene expression markers |
| **MRI-derived features** | Quantitative imaging features extracted from pre-treatment MRI scans |

Data preprocessing includes imputation of missing values, feature scaling, and dimensionality reduction to remove redundant or low-variance features.

---

## Methods

### Preprocessing
- Imputation of missing values across clinical and imaging features
- Feature scaling and normalisation
- Dimensionality reduction to eliminate redundant features

### Model
- Multilayer Perceptron (MLP) regressor trained to predict continuous relapse-free survival time
- Hyperparameter tuning to optimise architecture and learning dynamics
- Regularisation techniques (e.g. dropout, weight decay) to reduce overfitting

### Data Strategies
Four data strategies were explored to address the challenges of limited and imbalanced training data:

1. **Baseline** — no augmentation, raw training set only
2. **Gaussian noise augmentation** — synthetic samples generated by adding noise to existing training examples
3. **SMOTER regression oversampling** — adapted SMOTE technique for continuous targets, generating synthetic samples in underrepresented output regions
4. **SMOTER + forced feature inclusion** — SMOTER oversampling with guaranteed inclusion of key biological markers (ER status, HER2 status, gene expression features)

---

## Results

| Run | Data Strategy | MAE |
|-----|---------------|-----|
| 1 | Baseline (no augmentation) | 24.43 |
| 2 | Gaussian noise augmentation | 27.26 |
| 3 | SMOTER regression oversampling | 24.63 |
| 4 | SMOTER + ER, HER2 & Gene feature inclusion | 25.92 |

*MAE = Mean Absolute Error (lower is better). Units are in days of relapse-free survival.*

**Key takeaways:**
- The **baseline model** achieved the lowest MAE, suggesting that naive augmentation can introduce noise that harms generalisation.
- **SMOTER** came close to the baseline, offering a modest benefit for handling output imbalance without significantly degrading performance.
- **Forcing inclusion of biological markers** (ER, HER2, gene features) slightly increased error, indicating that while these features are clinically important, their integration requires careful tuning.
- Results highlight the difficulty of data augmentation in small, high-dimensional clinical datasets — a common challenge in medical ML.

## Notes

- This project was completed as part of **MSc Machine Learning in Science** coursework.
- SMOTER (Synthetic Minority Over-sampling Technique for Regression) is an extension of the classification-focused SMOTE algorithm, adapted for continuous target variables with uneven output distributions.
- MAE was chosen as the primary evaluation metric due to its interpretability in a clinical context — errors are expressed directly in units of survival time.
